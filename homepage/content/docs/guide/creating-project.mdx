# 创建项目详解

本指南将介绍如何使用 Vibe CLI 创建不同类型的 SaaS 项目，包括各种配置选项和最佳实践。

## 🎯 创建方式

交互式命令，系统会引导你完成所有配置：

```bash
vibe create my-app
```

## 其他命令

### run

```bash
vibe run xxx
```

1. 先执行 `./scripts/xxx.sh` 或 `./scripts/xxx.ts` 的脚本
2. 如果scripts中没有， 则去 package.json 中找 对应的 npm 的script 来执行

### add

```bash
vibe add [module]
```

目前只有几个:

事实上都是一个对应的 ts 脚本

```bash
vibe add package
[] axios
[] ai-sdk
```

```bash
vibe add database
[] drizzle:libsql
[] supabase
```

```bash
vibe add workflow
[] package_publish_on_github
[] vercel
```

```bash
vibe add services
[] user_manager
[] mail
[] payment
```

安装 tailwind v4。

```bash
vibe add ui
[] tailwind:v4
[] shadcn
[] icon:tabler
```

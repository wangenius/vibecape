# Getting Started

本指南将帮助你安装 Vibecape 并完成初始配置。

## 系统要求

| 平台 | 要求 |
|------|------|
| **macOS** | 12.0 (Monterey) 或更高 |
| **Windows** | Windows 10 或更高 |
| **Linux** | Ubuntu 20.04 或同等发行版 |
| **存储** | 至少 500MB 可用空间 |
| **内存** | 建议 8GB RAM |

## 安装

### macOS

1. 下载最新的 `.dmg` 文件：[Releases](https://github.com/wangenius/vibecape/releases/latest)
2. 打开 DMG 文件，将 Vibecape 拖入 Applications 文件夹
3. 首次启动可能需要右键选择「打开」以绕过 Gatekeeper

:::info
如果遇到「无法打开，因为无法验证开发者」的提示，请在系统偏好设置 → 安全性与隐私中点击「仍要打开」。
:::

### Windows

1. 下载最新的 `.exe` 安装程序
2. 运行安装程序，按提示完成安装

### Linux

1. 下载 `.AppImage` 或 `.deb` 包
2. AppImage: `chmod +x Vibecape.AppImage && ./Vibecape.AppImage`
3. Deb: `sudo dpkg -i vibecape.deb`

## 配置 AI 服务商

首次启动后，需要配置至少一个 AI 服务商：

:::warning
请妥善保管你的 API Key，不要将其分享给他人或提交到公开仓库。
:::

1. 打开 **设置** → **AI 服务商**
2. 添加服务商（如 OpenAI、DeepSeek、Google 等）
3. 填入 API Key 和 Base URL（如需要）
4. 添加并选择要使用的模型

支持的服务商：
- OpenAI / Azure OpenAI
- DeepSeek
- Google (Gemini)
- xAI (Grok)
- 302.AI
- 任何 OpenAI 兼容 API

## 数据存储

Vibecape 使用本地存储，数据目录结构：

```bash
~/vibecape/
├── config.json          # 应用配置
├── app.db               # AI 配置数据库
├── chat.db              # 全局聊天记录
└── workspaces.json      # 工作区索引

{docs_root}/             # 文档根目录（可配置）
├── {workspace_id}/      # 工作区目录
│   ├── config.json      # 工作区配置
│   ├── docs.db          # 文档数据
│   ├── chat.db          # 项目聊天
│   └── llm.txt          # AI 提示词
└── ...
```

## 创建工作区

1. 点击侧边栏的 **+** 按钮
2. 输入工作区名称
3. 开始创建文档

每个工作区拥有独立的：
- 文档树
- 聊天记录
- AI 上下文提示词 (`llm.txt`)

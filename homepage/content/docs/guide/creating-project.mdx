### `vibe run`

Execute scripts defined in your project.

```bash
vibe run [script-name]
```

**How it works:**

1. Checks for a matching script in `./scripts/[script-name].sh` or `./scripts/[script-name].ts`.
2. Fallback: Checks your `package.json` scripts.

### `vibe add`

Inject capabilities into your project. Features are modular and "drop-in".


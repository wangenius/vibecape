# Introduction

**Vibecape** 是一款 AI 驱动的本地优先文档编辑器，专为需要 AI 协作写作但又注重隐私和数据控制的用户设计。

## 为什么选择 Vibecape？

- **本地优先**: 所有数据存储在本地，你的文档永远属于你
- **AI 原生**: 深度集成多种 AI 模型（OpenAI、DeepSeek、Google、xAI 等）
- **工作区管理**: 统一的文档根目录，轻松备份和迁移
- **富文本编辑**: 基于 Tiptap 的现代编辑器，支持 Markdown

## 核心功能

| 功能 | 描述 |
|------|------|
| **AI 对话** | 与 AI 进行上下文感知的对话，支持流式输出 |
| **多模型支持** | 配置多个 AI 服务商和模型，按需切换 |
| **工作区** | 每个项目独立的文档库、聊天记录和 AI 提示词 |
| **llm.txt** | 为每个工作区定制 AI 上下文提示词 |
| **树形文档** | 支持父子层级的文档组织结构 |
| **OSS 集成** | 可选的云存储图片上传（阿里云、腾讯云、S3） |

## 快速开始

1. **[下载最新版本](https://github.com/wangenius/vibecape/releases/latest)** (macOS / Windows / Linux)
2. 安装并启动应用
3. 配置 AI 服务商（设置 API Key）
4. 创建你的第一个工作区

## ğŸ¯ Create Your First Project

### 1. Interactive Creation

Run the command and follow the prompts:

```bash
vibe create
```

**You will be asked to:**

- **Name your project**: e.g., `my-ai-app`
- **Pick a template**: Start with `ai-saas` for a full stack.
- **Select tools**: TypeScript (Recommended), Tailwind CSS (Recommended).

### 2. Quick Command

If you know what you want, skip the prompts:

```bash
vibe create my-ai-app --template=ai-saas --typescript --tailwind
```

## ğŸš€ Launch

### 1. Enter Directory

```bash

```

### 2. Install Dependencies

```bash

```

### 3. Start Dev Server

```bash
vibe run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see your app live!

## âš¡ Add Capabilities

Enhance your app immediately with `vibe add`.

### Add Authentication

```bash
vibe add auth --provider=clerk
```

_Installs Clerk, configures middleware, and adds login pages._

### Add Payments

```bash
vibe add payments --provider=stripe
```

_Sets up Stripe webhooks, pricing tables, and subscription management._

## ğŸ”§ Environment Setup

When adding integrations, `vibe` will generate a `.env.local` template.

```bash
cp .env.example .env.local
```

Fill in your keys:

```bash title=".env.local"
# Clerk Auth
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_test_...
CLERK_SECRET_KEY=sk_test_...

# Database
DATABASE_URL=postgresql://...
```

:::warning Security
Never commit `.env.local` to Git. Keep your API keys secure.
:::

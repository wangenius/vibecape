# 什么是 vibecape

vibecape 是一个 面向 AI vibecoder 的CLI开发工具。

## 用户

AI native coder

Product Creator

## 主要特性

- 基于场景的模版快速集成（包括对应模版的文档， 用于AI 开发）
- 深度集成AI，CLI本身可以使用 Agent
- 可拆式 模块化 功能
- 集成 node

## 一分钟上手

```bash
# 安装 vibecape
npm install -g vibecape

# 创建新项目
vibe create my-app

# 项目开始
vibe run dev
```
